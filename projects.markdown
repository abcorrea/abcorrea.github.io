## [Powerlifted](https://github.com/abcorrea/powerlifted) ##

Powerlifted is a classical planner using lifted representations and heuristic
search to solve planning problems. The core idea of Powerlifted is to use query
optimization and other database techniques to perform lifted planning more
efficiently.

You can find more information about the planner and relevant publications [at the GitHub repository](https://github.com/abcorrea/powerlifted).
